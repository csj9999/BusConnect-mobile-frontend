# BusConnect Mobile Application

## Overview
This repository contains the **mobile frontend application** for the **BusConnect Smart Transportation System**.
The application is developed using **Expo and React Native** and provides mobile interfaces for both **drivers and passengers**.

The mobile app communicates with backend and machine learning services to support real-time tracking, booking, analytics visualization, and system interaction.

---

## Key Features

### Driver Side
- View assigned routes and trips
- Real-time location sharing
- Passenger pickup visibility
- Driver-related insights and notifications

### Passenger Side
- Bus tracking and ETA viewing
- Booking and trip management
- Location-based services
- Travel history and analytics visualization
